# ng2-complexity
:metal: A complete Angular 2 project for TypeScript built with SystemJS, Gulp, Bootstrap 4, PostCSS, TSLint, Karma, Jasmine

## Quick start
```bash
git clone https://github.com/NN77/ng2-complexity.git
cd ng2-complexity
npm install
npm start
```
